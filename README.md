## Installation

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build the project

```bash
npm run build
```

### Run the project in production mode

```bash
npm run start
```
